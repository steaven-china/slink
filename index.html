slink - Secure SSH
Connection Manager

Encrypt your SSH credentials locally. Connect with a single command. No need to remember IPs, ports, or usernames ever again.

GitHub Get Started
Installation Usage Commands Security Configuration GUI

Features

🔐

Encrypted Storage

AES-128-CBC + HMAC via Fernet. PBKDF2-HMAC-SHA256 with 1M iterations.

🏷️

Aliases

Assign multiple aliases to one host. sli connect www just works.

🔗

Jump Hosts

Multi-level SSH proxy chains. sli jump-list bastion to discover targets.

📦

JSON Import/Export

Backup, migrate, or script your configs. Single-file .json quick-connect.

👥

Multi-User Isolation

Per-user encrypted stores on shared bastions via SLINK_USER.

🔑

Password Rotation

Change your master password anytime. Old data transparently upgrades.

Quick Start

1. Install
pip install -r requirements.txt
python setup.py install
2. Initialize
sli init
3. Add a host
sli add web1 -h 192.168.1.100 -u root -a www -a prod
4. Connect
sli connect web1
sli connect www

Commands

sli initSet master password
sli add <name>Add a host
sli listList all hosts
sli show <name>Host details
sli connect <name>SSH connect
sli edit <name>Edit host
sli rm <name>Remove host
sli importFrom SSH config
sli exportBackup to JSON
sli passwdRotate password
sli jump-listDiscover targets
sli namesNo-password list
