Configuration

Config Directory

Default location:

~/.slink/

On Windows this resolves to %USERPROFILE%\.slink\.

File Layout

~/.slink/
├── hosts.enc # Encrypted host database
├── salt # PBKDF2 salt (0o600)
├── .show_direct # Plaintext name index (0o600)
└── .lock # Advisory lock file

~/.slink/users/alice/ # When SLINK_USER=alice
├── hosts.enc
├── salt
├── .show_direct
└── .lock

Multi-User Isolation

Set the environment variable to switch config directories:

export SLINK_USER=alice
sli list # reads ~/.slink/users/alice/

Useful on shared bastion hosts where multiple operators need separate encrypted stores.

SSH Config Import

Import existing ~/.ssh/config entries:

sli import
sli import --file /path/to/custom_ssh_config

Only Host, Hostname, User, Port, and IdentityFile are imported. Passwords are prompted afterward.

Shell Completion

Add to your shell profile:

# Bash
eval "$(_SLI_COMPLETE=bash_source sli)"

# Zsh
eval "$(_SLI_COMPLETE=zsh_source sli)"

# Fish
_SLI_COMPLETE=fish_source sli | source
